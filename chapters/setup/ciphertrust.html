Deploy CipherTrust Manager

To use the samples in this ebook, you would require Thales CipherTrust Manager Community Edition installed and reachable from your orchestrator server. You would also need some of the many connectors provided by Thales CipherTrust Data Security Platform.

In this chapter, we will work our way to setting up CipherTrust Manager in cloud, VM or servers. We will be using the free-forever CipherTrust Manager Community Edition.

  • Problem

    Deploy a single node CipherTrust Manager instance in public or private cloud or in a Virtual Machine

  • Solution

    Solution depends on various factors like weather you want to deploy the CipherTrust Manager instance in public clouds like Azure, Google, AWS, or private cloud like VSphere or on a Virtual Machine.

    We will be covering some recipes in this chapter where you can deploy CipherTrust Manager manually, using tools like PowerShell or Terraform.

  • In cloud wih Terraform

    To deploy CipherTrust Manager on Azure, Google, or Azure clouds using the respective marketplace images, refer to the documentation and corresponding Terraform Scripts here at GitHub


  • In cloud wih PowerShell

    To deploy CipherTrust Manager on Azure using the respective marketplace images, refer to the sample PowerShell Script here at GitHub Gist


  • Deploy Manually

    To deploy manually on public or private clouds refer to ThalesDocs
